First-time setup: Connect to Firebase for cloud storage & login.

Step 1: Create a Firebase Project

  1. Go to console.firebase.google.com
  2. Click "Add project" → name it "LayFive" → Continue
  3. Turn off Google Analytics (optional) → Create Project

Step 2: Enable Email Login

  1. In your project, go to Build → Authentication → Get Started
  2. Click "Email/Password" → Enable → Save

Step 3: Create Database

  1. Go to Build → Firestore Database → Create Database
  2. Choose "Start in test mode" → pick a location → Enable

Step 4: Get Config

  1. Go to Project Settings (⚙️ gear icon)
  2. Under "Your apps" → click Web (</>) → Register app
  3. Copy the firebaseConfig object and paste below:
